### Виды календаря
- **Месяц** - сетка с днями
- **Неделя** - почасовое расписание
- **День** - детальное расписание на день
- **Список** - все события списком

### Возможности

1. **Создание событий**
   - Название, описание
   - Дата и время (начало/конец)
   - Повторение (ежедневно, еженедельно, ежемесячно)
   - Напоминания (за 10мин, 1час, 1день)
   - Цветовые метки

2. **Интеграция с модулями**
   - Модули могут создавать события в календаре
   - Пример: Habit Tracker создаёт напоминания
   - Пример: Study Assistant добавляет экзамены
   - Пример: Sleep Tracker добавляет напоминание лечь спать

3. **Уведомления**
   - Push-уведомления
   - Email-напоминания
   - Уведомления в чате с ИИ

4. **ИИ интеграция**
   - "Что у меня сегодня?"
   - "Добавь встречу на завтра в 15:00"
   - "Напомни мне позвонить маме в пятницу"
   - "Покажи мои события на следующей неделе"

### API для модулей

Модули могут взаимодействовать с календарём:

```python
# Создать событие
await calendar.create_event(
    title="Тренировка",
    start="2025-11-15T10:00:00",
    end="2025-11-15T11:00:00",
    reminder=10  # за 10 минут
)

# Получить события
events = await calendar.get_events(
    start_date="2025-11-15",
    end_date="2025-11-15"
)

# Создать повторяющееся событие
await calendar.create_recurring_event(
    title="Напоминание: лечь спать",
    time="22:00",
    repeat="daily"  # каждый день
)
```

### Синхронизация

- Экспорт в Google Calendar (iCal формат)
- Импорт из Google Calendar
- Двусторонняя синхронизация (опционально)
